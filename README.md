# MagicCal — Web 版本（已替换）

工作区已重建为一个**桌面浏览器可运行**的网页版魔术计算器（外观为手机计算器）。

快速使用
1. 打开 `index.html`（双击或用浏览器打开）。
2. 或运行：`python -m http.server 8000`，然后在浏览器打开 `http://localhost:8000/`。

主要行为（演示流程）
- 输入随机 4 位数 → `+` → 输入随机 5 位数 → `=` 显示 A+B。
- 再按 `+` → 点击 `观众模式` 并背向观众；观众随意按键但界面不会显示他们按的数字，而是显示“补数”（目标 - 已有和）。
- 再按 `=` → 总和即为预设的时间数字（例如 `202602181734`），并显示可读格式。

文件位置
- `index.html`, `styles.css`, `app.js` — 浏览器端代码（根目录）。
